# Review Service


## ローカル環境

```
Postman → API → ScalarDB Cluster → MySQL
```

ScalarDB Cluster & MySQL は Docker で手動起動済み

## コマンド

```bash
./gradlew build     # ビルド
./gradlew bootRun   # アプリ起動
```
